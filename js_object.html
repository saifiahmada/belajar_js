in JS, object is a king, if you understand object, you understand JS

In JS, almost "everything" is ab object

  • Boolean can be object(if defined with the new keyword)
  • Numbers can be object(if defined with the new keyword)
  • String can be object(if defined with the new keyword)
  • Dates are always objects
  • Maths are always objects
  • Regular expressions are always objects
  • Arrays are always objects
  • Functions are always object
  • Objects are always objects

All javascript values, except primitives are objects

JS Primitives, a primitives value is a value that has no properties or methods

JS defines 7 types of primitives data types

  • string
  • number
  • boolean
  • null
  • undefined
  • symbol
  • bigint

Objects are Variables

A JS object is a collection of named values

It is a common practice to declare objects with the const keyword

Object Properties

The named values in JS objects are called properties

Object Methods

Methods are action that can be performed on objects

Object properties can be both primitives values, other objects, and function

An object method is an object property containing a function definition

Creating a JS Object

  • Create a single object, using an object literal
  • Create a single object, with the keyword new
  • Define an object constructor and then create objects of the constructed type
  • Create an object Object.create()

Using an object Literal

Example : const person = {firstName: "John" , age:50}

Using the JS Keyword new

Example : const person = new Object()
