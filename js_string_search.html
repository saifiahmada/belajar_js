The indexOf() method returns the position of the first occurence of a specified text

The lastIndexOf() method returns the position of the last occurrence of a specified text

indexOf() dan lastIndexOf() mengembalikan nilai -1 jika tidak ditemukan

indexOf() dan lastIndexOf() menerima parameter kedua, sebagai posisi awal pencarian

search() method returns the position of the first occurence of a specified text in a string

match() search a string for "ain"

If a reguler expression does not include the g midifier (to perform a global search), the match() method will return only the first match in the string

Example global search case-insensitive for ain

includes() method returns true if a string contains a specified value

includes(searchValue)

includes(searchValue, start)

startsWith() method returns true if a string begins with a specified value, otherwise false

startsWith(searchValue)

startsWith(searchValue,start)

endsWith() method returns true if a string ends with a specified value, otherwise false

endsWith(searchValue)

endsWith(searchValue,length)
