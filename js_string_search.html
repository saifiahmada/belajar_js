The indexOf() method returns the position of the first occurence of a specified text

The lastIndexOf() method returns the position of the last occurrence of a specified text

indexOf() dan lastIndexOf() mengembalikan nilai -1 jika tidak ditemukan

indexOf() dan lastIndexOf() menerima parameter kedua, sebagai posisi awal pencarian

search() method returns the position of the first occurence of a specified text in a string
