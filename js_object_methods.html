What is this

in JS, the 'this' keyword refers to an object

Which object depends on how this is being invoked (used or called)

The this keyword refers to different objects depending on how it used

  • In an object method, 'this' refers to the object
  • Alone, 'this' refers to the global object
  • In a function, 'this' referes to the global object
  • In a function, in strict mode, 'this' is undefined
  • In an event, 'this' refers to the element that received the event
  • Method like call(), apply(), and bind() can refer 'this' to any object

NOTE:'this' is not a variable, It is a keyword, You cannot change the value of 'this'

JS methods are actions that can be performed on object

A JS method is a property containing a function definition

Methods are functions stored as object properties

Accessing Object Methods

You can access an object method with the following syntax:

objectName.methodName()

How to display JS Objects

document.getElementById("demo").innerHTML = person

Displaying a JS object will output -> [Object Object]

Some common solutions to display JS objects are:

  • Displaying the object properties by name, ex : person.city
  • Displaying the object properties in a loop, ex: for(let x in person)
  • Displaying the object using Objects.values(), ex: const myArray = Object.values(person)
  • Displaying the object using JSON.stringify(), ex: myString = JSON.stringify(person) => results JSON notation {"name":"John","age":50}
